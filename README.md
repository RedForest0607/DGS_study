# DGS
## TL;DR;  
DGS는  넷플릭스에서 `graphQL`을 보다 편리하게 사용하기 위해서 만든 프레임 워크로써 다음과 같은 기능을 제공합니다.
- 어노테이션 기반의 스프링 부트 모델 제공
- 유닛 테스트를 위한 쿼리테스트 기능 제공
- 스키마를 통해서 타입 생성
- WebScokets SSE를 통한 GraphQL 구독
- GraphQL을 통한 파일 업로드
